Super Tic Tac Toe
An advanced, interactive Super Tic Tac Toe game built with HTML, CSS, and JavaScript. Play against a friend or challenge the CPU with three difficulty levels. Track your wins, best records, and game statistics with a built-in leaderboard.

Features
Game Modes:
Play vs CPU (Easy, Medium, Hard)
Play with Friend (local two-player)
Super Board: 9 mini Tic Tac Toe boards in a 3x3 grid
Meta Win Logic: Win mini boards to claim them, win 3 in a row to win the game
Leaderboard:
Tracks wins, draws, best records (fewest moves), recent wins
Persistent stats using localStorage
Game Controls:
New Game, Reset Score, Change Mode, Undo (PvP only)
Responsive Design: Works on desktop and mobile
How to Play
Clone the repository:
git clone <your-repo-url>
Open super-tictactoe.html in your browser.
Select your game mode and start playing!
File Structure
super-tictactoe.html — Main game interface
super-tictactoe-styles.css — Game styling
super-tictactoe-script.js — Game logic and leaderboard
